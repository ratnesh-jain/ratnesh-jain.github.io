A Jumpbar using SwiftUI with TCA.

As in the video above, you can see the app is using a bottom toolbar style view to display directory location as in jumpbar. This is using the SwiftUI's safeAreaInset view modifier for to display the jumpbar at the bottom.

struct AppRootView: View {
    var body: some View {
        DirectoryItemsView(...)
            .safeAreaInset(.bottom) {
                JumpbarView(...)
            }
    }
}
